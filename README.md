# grfx

## Install

```bash
npm i grfx
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'grfx'
import 'grfx/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
